# Calculator App

A simple calculator web application built using HTML, CSS, and JavaScript. This project demonstrates basic frontend development skills and JavaScript logic implementation.

## 🚀 Features
- Perform basic arithmetic operations (+, -, *, /)
- Clean and user-friendly interface
- Responsive layout

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
